Como usar

1. Preencha os dados do sistema no formulário ao lado.

2. Clique em "Adicionar Sistema" para salvar.

3. Para acessar um sistema, clique no botão "Acessar".

4. Use a busca para localizar rapidamente um sistema.

5. Faça backup regular dos seus dados usando a seção de backup.

6. Os dados são armazenados localmente no seu navegador.
